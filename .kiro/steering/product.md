# Product

Personal portfolio and technical blog for Rafal M Cichon, a Senior Digital Growth & Web Engineer.

## Purpose
- Showcase professional work (projects, case studies)
- Share technical notes and lessons learned
- Demonstrate expertise in Shopify UX, CRO, technical SEO, performance, and accessibility
- Provide resume and contact information

## Content Types
- **Projects**: Case studies with measurable outcomes (conversion lifts, sales increases)
- **Notes**: Short technical write-ups on specific topics
- **Log**: Wins, losses, and lessons learned
- **Resume/Experience**: Professional highlights and background

## Design Philosophy
- Clean HTML with single CSS file
- Strong accessibility (semantic HTML, keyboard navigation, ARIA)
- Strong SEO basics (structured data, meta tags, sitemap)
- Content written in Markdown
- LLM-friendly content structure (clear answers first, then context)
- Performance-focused (Core Web Vitals, optimized images, minimal scripts)
